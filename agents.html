Agents
5 Specialized Agents

Each agent owns a specific domain and executes its skills autonomously.

E

Extractor Agent

Paper Analysis & Parameter Extraction

Reads research papers (PDF) and extracts electrode parameters, material properties, and simulation conditions automatically using vision-capable LLM.

rag-knowledge
G

Generator Agent

Electrode Structure Generation

Creates 3D electrode microstructures using particle packing algorithms (RSA, Force-Biased, FCC, BCC). Supports spherical, custom STL, and bimodal particles with polydisperse size distributions.

geometry-sphere geometry-custom geometry-gap
F

Fabricator Agent

3D Visualization & Export

Renders electrode structures in Blender with instancing and size-based coloring. Exports to STL/DXF for simulation. Simulates calendering via rigid/soft body physics.

visualize-blender export-stl export-dxf simulate-calendering simulate-softbody capture-viewport
V

Validator Agent

Structural Analysis & Validation

Runs 8 MATLAB-based validation metrics on generated electrodes: porosity, tortuosity, PSD, surface area, connectivity, two-point correlation, voxelization, and mesh quality.

validate-porosity validate-tortuosity validate-psd validate-surface validate-connectivity validate-twopoint validate-voxel validate-mesh
S

Simulator Agent

Electrochemical Simulation

Drives COMSOL Multiphysics via LiveLink for MATLAB. Imports DXF geometry, sets up materials/physics, generates mesh, runs solver, and exports results (voltage curves, contour plots).

simulate-halfcell simulate-fullcell simulate-fullcell-1d simulate-dis simulate-eis
Skills
24+ Modular Skills

Each skill is a self-contained unit with defined inputs, outputs, and deterministic execution.

Geometry (3 skills)

Geometry

geometry-sphere

Spherical particle packing. Single size or polydisperse (d10/d50/d90). RSA, Force-Biased, FCC, BCC algorithms.

Geometry

geometry-custom

Custom STL particle packing. Single or multiple STL mixing. Supports polydisperse sizing with non-spherical shapes.

Geometry

geometry-gap

Two-type (A-B) particle packing with controlled gap distance. Fixed gap or gap distribution (d10/d50/d90).

Visualization & Export (6 skills)

Visualization

visualize-blender

Blender 3D rendering with instancing (1000x speedup), size-based coloring, and real-time viewport.

Export

export-stl

Export particles to STL format. Individual meshes combined into single file for MATLAB validation.

Export

export-dxf

Export to DXF format for COMSOL import. 2D cross-sections or 3D wireframes.

Physics

simulate-calendering

Rigid body gravity settling for electrode densification. Blender Bullet physics engine.

Physics

simulate-softbody

Soft body deformation simulation. Particles flatten under compression for realistic calendering.

Capture

capture-viewport

Capture Blender viewport or render to PNG for documentation and reporting.

Validation (8 skills)

Validation

validate-porosity

Analytical and voxel-based porosity calculation. Compares target vs achieved volume fraction.

Validation

validate-tortuosity

Tortuosity via TauFactor MATLAB App. Directional tau values and percolation status.

Validation

validate-psd

Particle size distribution analysis. Histogram + d10/d50/d90 comparison with target.

Validation

validate-surface

Specific surface area calculation from STL mesh triangles.

Validation

validate-connectivity

Particle connectivity and percolation network analysis.

Validation

validate-twopoint

Two-point correlation function for spatial homogeneity assessment.

Validation

validate-voxel

Voxelization + 4-panel visualization (3D mesh, XY/XZ/YZ projections).

Validation

validate-mesh

STL mesh quality analysis: triangle count, vertex count, aspect ratio, edge quality.

Simulation (5 skills)

Simulation

simulate-halfcell

COMSOL half-cell electrochemical simulation. DXF import, materials, physics, mesh, solve, post-process.

Simulation

simulate-fullcell

3D full cell (anode + cathode). N/P ratio optimization and cell voltage tracking.

Simulation

simulate-fullcell-1d

Simplified 1D full cell for quick validation and P2D comparison.

Simulation

simulate-dis

Diffusion-Induced Stress. Couples electrochemistry with solid mechanics for particle cracking analysis.

Simulation

simulate-eis

Electrochemical Impedance Spectroscopy. Analytical model with Nyquist and Bode plots.
